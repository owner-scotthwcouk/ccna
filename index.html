Network Fundamentals

Domain 1

Understand OSI/TCP-IP models, cabling, speed/duplex negotiation, and how frames, packets, and segments move through a network.

  • Explain OSI layers with real device examples.
  • Describe Ethernet standards and cabling choices.
  • Compare TCP vs UDP and when to use each.

Study Checklist

  • Map common protocols to OSI layers (HTTP, ARP, DNS, DHCP).
  • Practice subnetting using /24 to /30 ranges.
  • Know collision vs broadcast domains.

IPv4/IPv6 Addressing

Domain 1

Cover binary math, subnetting, VLSM, IPv6 structure, and address types including global unicast, link-local, and multicast.

  • Summarize IPv4 addressing and subnetting steps.
  • Explain IPv6 address notation and compression rules.
  • Differentiate unicast, multicast, and anycast.

Lab Ideas

  • Create an addressing plan for a 3-site network.
  • Assign IPv6 link-local and global unicast addresses.
  • Validate with show ip interface outputs.

Switching Concepts

Domain 2

Dive into MAC address tables, forwarding logic, VLANs, trunking, and spanning tree to prevent loops.

  • Describe how switches learn and age MAC addresses.
  • Configure VLANs and 802.1Q trunks.
  • Interpret STP port roles and states.

Key Commands

  • show mac address-table
  • show spanning-tree
  • switchport mode trunk

Wireless Networks

Domain 2

Explore wireless standards, RF fundamentals, AP modes, WLC roles, and roaming strategies.

  • Compare 802.11 a/b/g/n/ac/ax features.
  • Explain SSIDs, BSS, ESS, and roaming.
  • Understand authentication and encryption options.

Study Prompts

  • Identify channel plans for 2.4GHz vs 5GHz.
  • Explain CAPWAP roles and WLC components.
  • List common wireless security weaknesses.

Routing Concepts

Domain 3

Learn about routing tables, static vs dynamic routing, and the administrative distance/metrics used to pick paths.

  • Describe packet forwarding with longest prefix match.
  • Configure default routes and floating static routes.
  • Interpret routing table codes and next-hop logic.

Checklist

  • Build a static routing lab across 3 routers.
  • Verify with show ip route.
  • Explain how routing loops occur and avoid them.

Single-Area OSPF

Domain 3

Understand OSPF neighbor states, LSA types, DR/BDR elections, and cost calculations for single-area deployments.

  • Configure OSPFv2/v3 with network statements.
  • Explain OSPF adjacency requirements.
  • Use passive interfaces and optimize convergence.

Command Drill

  • show ip ospf neighbor
  • show ip ospf interface
  • router ospf 1 + network

DHCP & DNS

Domain 4

Master DHCP operations, DHCP relay, DNS hierarchy, record types, and how clients resolve names.

  • Describe the DHCP DORA process.
  • Configure DHCP pools and exclusions.
  • Explain DNS A/AAAA/CNAME/MX records.

Practice

  • Build a DHCP server on a router and test leases.
  • Use nslookup to validate DNS responses.
  • Map DNS troubleshooting steps.

NAT & NTP

Domain 4

Understand NAT types, translation tables, and time synchronization with NTP for log accuracy.

  • Compare static, dynamic, and PAT.
  • Verify translations with show ip nat.
  • Configure NTP clients and servers.

Key Reminders

  • Know inside vs outside interfaces.
  • Set NTP authentication where possible.
  • Document time drift troubleshooting steps.

Device Security

Domain 5

Apply AAA concepts, secure management access, and implement password policies for network devices.

  • Configure local users and privilege levels.
  • Enable SSH and disable insecure services.
  • Use banners, exec timeouts, and logging.

Hardening Tasks

  • Implement enable secret and SSH keys.
  • Apply role-based CLI views (if available).
  • Verify with show run | section line vty.

Network Security

Domain 5

Cover ACLs, port security, DHCP snooping, and common attack vectors like spoofing or man-in-the-middle.

  • Build standard and extended ACLs.
  • Configure port security with sticky MACs.
  • Identify and mitigate common L2 attacks.

Checklist

  • Document ACL direction and wildcard masks.
  • Enable DHCP snooping and DAI.
  • Test ACLs using ping and traceroute.

Automation & Programmability

Domain 6

Learn about APIs, JSON/YAML, RESTCONF/NETCONF, and the difference between controller-based and device-by-device management.

  • Explain data formats and model-driven telemetry.
  • Use Postman or curl with REST APIs.
  • Describe benefits of SDN architectures.

Practice Ideas

  • Parse JSON from a sample API response.
  • Write a simple Python script to configure devices.
  • Document automation use cases in your lab.
